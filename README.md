# 株式会社ゆめみ フロントエンドコーディング試験

## ローカル環境セットアップ

1. Next.js セットアップ＆起動

```
cp .env.example .env.local
yarn
yarn dev
```

2. RESAS API の API キーを`.env.local`に設定してください  
   （参考 URL：https://opendata.resas-portal.go.jp）
